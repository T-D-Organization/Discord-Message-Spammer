## 📌 Features

- ✅ **Terminal Input** – Token aur Channel ID terminal se mangega (code mein put nahi karna)
- ✅ **Auto-Save Config** – Token & Channel ID `token.txt` mein save ho jayega
- ✅ **Millisecond Support** – 0.1 seconds (100ms) se lekar 60 seconds tak
- ✅ **3 Spam Modes**:
  - Single Message (Ek hi message baar baar)
  - Multiple Messages (Messages rotate honge)
  - Sequential Mode (Message 1,2,3,1,2,3...)
- ✅ **Spam Counter** – Kitni baar bhejna hai (0 = Infinite)
- ✅ **Rate Limit Handling** – Auto-wait agar Discord block kare
- ✅ **Progress Bar** – Real-time progress dikhega
- ✅ **Beautiful Terminal UI** – Boxed design with ASCII banner
- ✅ **Lightweight** – Sirf `requests` library chahiye
